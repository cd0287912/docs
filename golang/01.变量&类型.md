### Go 变量

在 go 中定义变量的方式有几种，定义全局变量不能使用省略格式

```go
  // 完整格式
  var name string = "yolo"
  // 类型推断
  var name = "yolo"
  // 省略
  name := "yolo"
  // 定义多个变量 - 类型推断
  var name, age = "yolo", 18
  name, age := "yolo", 20
  var (
    name string = "yolo"
    age  int
  )
```

定义常量使用 const 关键字

```go
  const PI float32 = 3.1415
  // 定义多个
  const (
    A = "A"
    B = 12
    C = false
    D = 'A'
  )
```
