### GO 流程控制

go 语言常用的流程控制有 if 和 for，而 switch 好 goto 主要是为了简化代码、降低重复代码而生的结构，注意 go 中没有 while 语句，可以使用 for 代替

#### if 语句

```go
	flag := true
	if flag {
		fmt.Println("flag = true")
	}
    // 变量的作用域不同
	if age := 30; age > 40 {
		fmt.Println("成年人")
	}
    score := 88
	if score >= 90 {
		fmt.Println("优秀")
	} else if score >= 80 {
		fmt.Println("良好")
	} else {
		fmt.Println("一般")
	}

	if score := 60; score > 90 {
		fmt.Println("A")
	} else if score > 80 {
		fmt.Println("B")
	} else {
		fmt.Println("C")
	}
```
