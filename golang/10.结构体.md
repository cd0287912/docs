### GO 结构体struct

使用type和struct来定义结构体。通俗来说就是自定义类型。

```go
// type 类型名 struct {
//   字段名 字段类型
//   字段名 字段类型
// }

type person struct {
	name string
	age  int
	sex  string
}

// 结构体实例化
var user person // 实例化
user.name = "张三"
user.age = 18
user.sex = "男"
fmt.Printf("%#v\n", user)

// 结构体实例化
var user = new(person)
user.name = "张三"
user.age = 18
user.sex = "男"
fmt.Printf("%#v\n", user)

// 结构体实例化
var user = &person{}
user.name = "张三"
user.age = 18
user.sex = "男"
fmt.Printf("%#v\n", user)
```
